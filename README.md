# du_cleaner
WANRING: Will remove files. Use at own risk.   
The script will remove directories in the target directory if they are old enough.  
If the target_directory contains a directory older than the specified time in the find-commands.  
Only directories that start with "run_" and contain the file ".done" will be removed  

# quick start  
./du_cleaner.sh [target_directory]  [second_target_directory]
